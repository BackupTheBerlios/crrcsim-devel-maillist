'copyrighttodo_jw' to it.

Here is a shellscript which lists dates and contributors to every file in
the current directory (glad to have hg again):

  +--------------------------------------------------------
  | #! /bin/sh
  | 
  | for n in *
  |   do
  |      if test -f $n
  |        then
  |          echo $n
  |          hg log -f -l 9999 --template "   {date|shortdate} {author|user}\n" $n
  |      fi
  | done
  +--------------------------------------------------------
