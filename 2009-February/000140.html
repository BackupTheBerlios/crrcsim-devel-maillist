Hi,

so I pushed my changesets, they partially clean up what I didn't like.

Furthermore, they provide sensible launch defaults for every model (except
mav, see below) and provide mostly backwards-compatible behaviour. So this
is the first version which provides a working launch for every model after
selecting it, without tweaking launch settings. No sports models/heli will
fall from the sky and no glider will sit on the grass.

I added the player-relative launch position to the launch default section in
the model file and to launch presets in crrcsim.xml. The new values are
handled by crrc_launch.cpp -- however, they are not editable using the GUI yet.

'mav' can't be launched in any way, it must be started from the ground. I'll
look into it -- it has been like this since some time, though...

I recognized that one of the recent changesets changed the player position
in such a way that when doing a ground start at Davis, the initial field of
view has changed a lot -- the player has grown taller some feet.

Jens
